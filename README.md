# Melanie Montague Personal Site

This is a simple static website built from the Markdown source in `source/melanie-profile-source.md`.

## Open The Site

Open `index.html` in a browser.

## Files

- `index.html` - page structure and content
- `resume.html` - web resume with print-friendly styling
- `styles.css` - BYU-inspired tennis visual design and animations
- `script.js` - scroll reveal animation and image fallback behavior
- `assets/` - place `Aaron&Me.JPEG` here if you want the local personal photo to appear
- `source/` - original Markdown source

No build step or package install is required.

## Image Notes

For the best result, place real personal photos in `assets/` and update the image paths in
`index.html`. The current site keeps a clean placeholder for `assets/Aaron&Me.JPEG` so the
homepage still looks intentional if that file is missing.
